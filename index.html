4.良好的沟通能力和团队协作能力。

初级区块链工程师

1.精通 C/C++/Python/Go/Nodejs 中至少两门语言，至少6年以上经验。

2.熟悉基于Git的团队开发模式。

3.在网络编程，密码学，分布式一致性算法，虚拟机，编译器以及数据库的开发经验者优先。

4.了解区块链共识机制，了解各种区块链产品和超级账本等 ，至少一种主流项目的底层架构。

5.具有良好的英语阅读能力，良好的沟通能力和团队协作能力。

高级区块链工程师

1.熟悉AI和Photoshop软件，能够根据UI设计稿制作网页；

2.熟悉HTML5，CSS，Javascript编程语言；

3.熟悉 vue / react 前端 MVVM 开发框架者优先有 SPA 类网站开发经验者优先;

4.熟悉响应式Web设计原理，具备实践经验；

5.能够熟练运用现有的技术，创造最佳的用户体验；

6.良好的英文阅读能力，能够顺畅阅读英文技术文档
